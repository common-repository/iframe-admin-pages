== Description ==

iFrame Admin Pages plugin for WordPress lets you quickly and easily setup new administration pages by displaying any page on the internet in an iframe. This basically means you can include your fave websites and social network profiles all in one place for quick access. Perfect if you spend a lot of time on your WordPress blog.

== Installation ==

No coding skills required to use this plugin and you can install as you would most plugins.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
